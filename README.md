Omega-Actions-Plugin
The plugin creates the sit, crouch, crawl.

Usage: Sitting : "Z" (90) ; Crouch : "Í" (226) ; Crawl : "U" (85).
The plugin is still under development. All ideas, advice and improvements are welcome.
